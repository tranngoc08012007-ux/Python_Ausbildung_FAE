# Phase 01 — Python Fundamentals & Professional Development Environment

## Overview

This phase establishes a strong foundation in Python programming while introducing the tools and workflows used in professional software development.

By the end of this phase, you will not only understand Python fundamentals but also know how to build projects using industry-standard practices such as virtual environments, Git, code formatting, and linting.

---

# Learning Objectives

After completing this phase, you will be able to:

* Set up a professional Python development environment.
* Work with virtual environments and package management.
* Follow clean coding practices using **Black** and **Flake8**.
* Use Git and GitHub with a professional workflow.
* Understand and apply Python fundamentals.
* Build small Python programs independently.

---

# Lessons

| Lesson | Topic                                | Description                                                                                                            |
| ------ | ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------- |
| 01     | Environment Setup                    | Install Python 3.11+, configure VS Code, install essential extensions, and learn basic terminal usage.                 |
| 02     | Virtual Environment & pip            | Create virtual environments, manage packages with `pip`, and understand `requirements.txt` and `requirements-dev.txt`. |
| 03     | Code Quality Tools                   | Format code with **Black**, lint with **Flake8**, configure `.editorconfig`, and enable format-on-save.                |
| 04     | Git Basics                           | Initialize repositories, commit changes, push to GitHub, use `.gitignore`, and write conventional commit messages.     |
| 05     | Git Advanced: Branch & Pull Requests | Work with feature branches, merge changes, resolve simple conflicts, and create Pull Requests.                         |
| 06     | Variables & Data Types               | Learn Python's built-in data types, type conversion, arithmetic operations, `input()`, and `print()`.                  |
| 07     | Strings                              | Manipulate strings using slicing, formatting, comparison, and common string methods.                                   |
| 08     | Lists & List Comprehension           | Work with lists, slicing, sorting, list methods, and list comprehensions.                                              |
| 09     | Control Flow                         | Write decision-making logic using `if`, `elif`, `else`, logical operators, and conditional expressions.                |
| 10     | Loops                                | Master `for` and `while` loops, `range()`, `enumerate()`, `zip()`, and loop control statements.                        |

---

# Skills Acquired

By the end of this phase, you will be able to:

* Configure Python and VS Code for professional development.
* Create and manage isolated Python environments using `venv`.
* Install and manage packages with `pip`.
* Maintain project dependencies using `requirements.txt`.
* Write clean, PEP 8-compliant Python code.
* Format code automatically using **Black**.
* Detect common coding issues with **Flake8**.
* Use Git effectively with branches and Pull Requests.
* Understand Python's core syntax and programming concepts.
* Solve basic programming problems using variables, strings, lists, conditionals, and loops.
